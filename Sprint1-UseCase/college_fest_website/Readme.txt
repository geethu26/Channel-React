College Fest Website

This project is a basic college fest website designed to promote college annual fests.

#Features:
Home Page with navigation bar
Events Page which has a list of all events
Registration Page for students
Gallery page to display images from past fests
Contact Page with contact information

##Technologies
HTML5
Bootstrap 5


##Folder Structure:
index.html
events.html
register.html
gallery.html
contact.html
style.css (optional)

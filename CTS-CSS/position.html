position:static

An element with position: static, is not posiyioned in any way; its always positioned according to normal flow of the page.

The div element has position: static

position:relative

An element with position:relative; is positioned relative to its normal position;

div element has position: relative

position:fixed

An element with position:fixed; is positioned relative to the viewport position, which means it always stays in the same place even when the page is scrolled.;

div element has position: fixed

position:absolute

An element with position:absolute; is positioned relative to the nearest positioned ancestor (i.e., the nearest ancestor element with a position value other than static). If there is no such ancestor, it is positioned relative to the initial containing block (usually the viewport).

div element has position: relative
div element has position: absolute
